Das Fundament

Was ist CLAUDE.md?

  • 1 Eine Markdown-Datei im Root eures Repos, die Claude sagt, wie es sich in eurem Projekt verhalten soll
  • 2 Versioniert im Repo. Euer ganzes Team teilt und reviewt sie in PRs
  • 3 Bleibt über Sessions bestehen. Einmal schreiben, Claude hält sich immer dran
  • 4 Kein System-Prompt. Sie ist projektspezifisch, kombinierbar und lebt in eurer Codebase
  • 5 Startet mit Karpathys Vorlage: claude install-skill https://github.com/forrestchang/andrej-karpathy-skills

Code als Schnittstelle

Was sind Skills?

Statt aufgabenspezifische Agenten zu bauen, behandelt Claude Code Code als „Schnittstelle zur digitalen Welt“. Ein universeller Agent, angereichert mit modularen Skills.

  • 1 Jeder Skill ist eine SKILL.md-Datei in einem Ordner, reines Markdown
  • 2 Installieren: claude skills add mattpocock/skills/grill-me
  • 3 Ausführen: /grill-me in der Konversation eingeben
  • 4 Eigene Skills schreiben dauert keine 10 Minuten
